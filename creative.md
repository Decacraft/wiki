# Le Monde Créatif
Ce plugin vous permet d'avoir une parcelle et d'y construire en mode créatif. Les parcelles font une taille de 100x100 et vous pouvez y faire ce que vous voulez (en respectant les règles bien entendu...).

### Les commandes de base
---
- ``/plot auto``: Permet de revendiquer la première parcelle disponible
- ``/plot home``: Permet de se téléporter à sa parcelle
- ``/plot visit [joueur]``: Permet de visiter le plot de quelqu'un
- ``/plot add [joueur]``: Autorise un joueur à construire si le propriétaire est en ligne.
- ``/plot trust [joueur]``: Autorise un joueur à construire même si le propriétaire est hors-ligne.
- ``/plot remove [joueur]``: Retire un joueur de sa percelle
- ``/plot clear``: Efface la parcelle sur laquelle vous êtes.

### Les commandes et avantage accessible via les grades
---
En tant que VIP vous avez la possibilité d'avoir un deuxième plot en monde créatif. Refaite la commande pour revendiquer une parcelle pour vous en attribuer une nouvelle.

- ``/plot home 2``: Permet de se téléporter à votre deuxième parcelle.
- ``/plot flag set weather [clear/rain]``: Permet de retirer ou mettre la pluie sur sa parcelle.
- ``/plot flag set time [0-24000]``: Permet de mettre le jour ou la nuit sur votre parcelle [(voir la correspondance des ticks par rapport au cycle jour/nuit)](https://minecraft.fandom.com/fr/wiki/Tick)

En tant que Master vous avez la possibilité d'avoir un troisième plot en monde créatif.

- ``/plot setbiome``: Permet de choisir le biome de votre parcelle.
